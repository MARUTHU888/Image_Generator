# 🎨 AI Image Generator Web Application

A full-stack AI-powered web application that generates high-quality images from text prompts using Stable Diffusion.

## 🚀 Features

- Text-to-Image generation using Stable Diffusion
- FastAPI backend API
- Responsive Tailwind CSS frontend
- Real-time image rendering
- Base64 image transfer from backend to frontend
- Clean and modern UI
- Local deployment support

## 🛠 Tech Stack

- Python
- FastAPI
- Stable Diffusion (Diffusers)
- PyTorch
- Tailwind CSS
- JavaScript
- HTML

## ⚡ How It Works

1. User enters a descriptive prompt.
2. Backend processes the request using Stable Diffusion.
3. Generated image is converted to Base64 format.
4. Image is sent to frontend and displayed instantly.

## 🎯 Example Prompt

"Ultra modern luxury villa with infinity pool, sunset lighting, ultra realistic, architectural photography, 4K"
